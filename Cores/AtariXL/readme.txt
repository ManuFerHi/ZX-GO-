1.1 (14-Feb-2017)

Añadida detección del modo de vídeo (RGB o VGA) desde BIOS. Ahora se activa el adecuado al arrancar el core.
Completada la emulación de Joystick en teclado numérico, teclas 5 y 2 sirven indistintamente para la dirección "abajo". 0 es fuego. Compatible para usar a la vez el joystick físico y el pad numérico sin interferencias. Ambos sirven para navegar por los listados de archivos.
Añadidas teclas Alt y Ctrl derechas como válidas también para el Master Reset.

- Ahora se pueden cargar cartuchos grandes (de hecho, enormes, hasta 1024K de cartucho, como el Space Harrier)!
- Ahora se pueden seleccionar 3 amplaciones de RAM más en el menú F12: 576K Compy, 576K Rambo y 1MB. (No probados aún. ¿Sabéis de algún juego o demo que los use?)
- He añadido al testigo LED el encendido mientras se cargan cartuchos. Para los grandes tarda un poco, y así sabemos que está cargando (antes no lo hacía).
